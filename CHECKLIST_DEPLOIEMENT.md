# ✅ Checklist de déploiement - BROQUES

Cochez chaque étape au fur et à mesure :

## 📋 Étape 1 : Vérification Supabase

- [ ] Tables créées (profiles, annonces, conversations, messages)
- [ ] RLS activé sur toutes les tables
- [ ] Bucket "photos" créé dans Storage
- [ ] URL et clés API notées

## 📋 Étape 2 : Configuration locale

- [ ] Fichier .env.local créé
- [ ] Variables d'environnement configurées
- [ ] Application testée en local

## 📋 Étape 3 : Git

- [ ] Dépôt Git initialisé
- [ ] Code commité
- [ ] Dépôt distant créé (GitHub/GitLab)
- [ ] Code poussé sur le dépôt distant

## 📋 Étape 4 : Vercel

- [ ] Compte Vercel créé
- [ ] Projet Vercel créé
- [ ] Dépôt connecté
- [ ] Variables d'environnement configurées
- [ ] Déploiement réussi

## 📋 Étape 5 : Tests

- [ ] Site accessible
- [ ] Inscription fonctionne
- [ ] Connexion fonctionne
- [ ] Création d'annonce fonctionne
- [ ] Upload de photos fonctionne
- [ ] Messagerie fonctionne
