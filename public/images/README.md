# Images

Placez votre image ici.

## Image des outils disponibles

Pour afficher l'image sur la page principale, ajoutez votre image dans ce dossier avec le nom :
- `outils-disponibles.jpg` (ou `.png`, `.webp`)

L'image sera automatiquement affichée dans la section "Des outils pour tous vos projets" de la page d'accueil.

**Format recommandé :**
- Format : JPG, PNG ou WebP
- Ratio : 16:10 (largeur:hauteur)
- Taille : 1200-1600px de largeur pour une bonne qualité
